## 插入排序的特点

- 时间复杂度是 `O(N^2)`, 空间复杂度是 `O(1)`
    - 如果传入的数据是增序排好的, 那么只需要 N-1 次的比较, 以及 0 次的交换
    - 如果传入的数据是降序排好的, 那么需要 N^2/2 次的比较, 以及 N^2/2 次的交换
    - 如果是乱序的, 大概需要 N^2/4 次的比较, 以及 N^2/4 次的交换
- 插入排序是稳定排序
- 插入排序比较适合元素较少的数组
- 插入排序适合基本已排序好的数组
- 插入排序常用于组合排序算法中, 用于排序较少元素的部分数组; 比如 cpp 里面的 `std::sort()` 以及 python
  里的 `timsort()`
