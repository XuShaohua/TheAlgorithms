# 项目排期

## 题目描述

项目组共有N个开发人员, 项目经理接到了M个独立的需求, 每个需求的工作量不同, 且每个需求只能由一个开发人员独立完成, 不能多人合作.
假定各个需求直接无任何先后依赖关系, 请设计算法帮助项目经理进行工作安排, 使整个项目能用最少的时间交付.

### 输入描述

- 第一行输入为M个需求的工作量, 单位为天, 用逗号隔开
    - 例如 X1 X2 X3 … Xm, 表示共有M个需求, 每个需求的工作量分别为X1天, X2天…Xm天
    - 其中 0<M<30, 0<Xm<200
- 第二行输入为项目组人员数量N

### 输出描述

最快完成所有工作的天数.
