# 找数字/找等值元素

## 题目描述

给一个二维数组 nums, 对于每一个元素 nums[i], 找出距离最近的且值相等的元素,

输出横纵坐标差值的绝对值之和, 如果没有等值元素, 则输出-1.

### 输入描述

- 输入第一行为二维数组的行
- 输入第二行为二维数组的列
- 输入的数字以空格隔开

备注:

- 针对数组 nums[i][j], 满足 0 < i ≤ 100, 0 < j ≤ 100
- 对于每个数字, 最多存在 100 个与其相等的数字

### 输出描述

数组形式返回所有坐标值.
